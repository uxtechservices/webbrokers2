# **App Name**: Minimalist Text Display

## Core Features:

- Text Input: Allow the user to input text.
- Text Display: Display the inputted text on a blank page.

## Style Guidelines:

- Primary color: Black (#000000) for text.
- Background color: White (#FFFFFF) for a clean, minimalist look.
- Body font: 'Inter', a sans-serif font for clear readability.
- Center the text on the page.
